# 千焦 / 卡路里自由转换

一个适合手机打开、可添加到 iPhone 主屏幕的轻量 PWA，用来做 `kJ` 和 `kcal` 的双向自由转换。

它不是传统“只给一个输入框”的计算器，而是把换算、标签阅读和能量感知放到同一个页面里：

- 双向输入：输入 `kJ` 或 `kcal` 都会实时联动
- 可视化能量尺：拖动滑杆快速感知区间
- 包装模式：把 `kJ` 标签翻译成更直觉的 `kcal` 读法
- 最近使用：本地保存最近换算值
- PWA 支持：可安装到 iPhone 主屏幕，支持离线首屏资源

## 适合的使用场景

- 看食品包装时，只标了 `kJ`，想快速换算成更熟悉的 `kcal`
- 日常控制饮食时，快速判断某个数值大概属于什么能量区间
- 想把工具加到 iPhone 主屏幕，随时点开直接用

## 换算公式

```text
1 kcal = 4.184 kJ
1 kJ ≈ 0.239 kcal
```

## 本地打开

项目是纯静态站点，不需要前端构建。

在项目目录里启动一个静态服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/
```

## iPhone 主屏幕使用

1. 用 Safari 打开站点。
2. 点击“分享”。
3. 选择“添加到主屏幕”。
4. 从主屏幕图标打开，即可获得更接近 App 的体验。

如果之前已经添加过旧版本，建议先删除旧图标，再重新添加一次，这样新的图标和启动页会生效。

## 设计方向

这版视觉不是走通用 SaaS 风格，而是刻意做成：

- 包装标签感
- 票据式信息层次
- 仪表盘式能量反馈
- 暖色纸张质感 + 高对比数字排版

这样它更像一个“口袋小工具”，而不是普通网页。
